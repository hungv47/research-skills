# Target Anti-Patterns

Ten named anti-patterns that kill target effectiveness. Each includes detection methods, real examples, and fixes.

---

## 1. Vanity Metrics Syndrome

**What it is:** Measuring metrics that look good but don't connect to business outcomes.

**Detection:**
- KRs focus on followers, likes, impressions, page views
- No clear link between the metric and revenue/retention
- Metric can be gamed easily (buy followers, run clickbait)

**Bad Example:**
```
KR: Increase Instagram followers from 10K to 50K
```

**Why it's bad:** Followers don't correlate with revenue. You can have 1M followers and zero customers.

**Good Example:**
```
KR: Increase qualified leads from social channels from 50/month to 150/month
```

**Fix:** For every vanity metric, ask "If this 5x'd tomorrow, would revenue change?" If no, find the metric that would.

---

## 2. The Feature Factory

**What it is:** Targets that measure feature launches instead of outcomes.

**Detection:**
- KRs are task completions: "Launch X", "Ship Y", "Build Z"
- Success is defined as "done" not "impact measured"
- No metrics tied to what happens after launch

**Bad Example:**
```
KR: Launch new onboarding flow by March 15
```

**Why it's bad:** You can ship on time and have zero impact. Shipping is the beginning, not the end.

**Good Example:**
```
KR: Increase onboarding completion rate from 40% to 60%
(Initiative: New onboarding flow)
```

**Fix:** Convert "Launch X" to "X causes Y improvement." The initiative is how; the KR is what changes.

---

## 3. Sandbagging

**What it is:** Setting targets that will definitely be achieved without extra effort.

**Detection:**
- 100% confidence in hitting the target
- Target is at or below current performance
- Historical trend would hit target without intervention

**Bad Example:**
```
Current MRR: $50K, growing 10% MoM naturally
KR: Achieve $55K MRR (10% growth over baseline)
```

**Why it's bad:** Targets should stretch the team. Guaranteed wins aren't goals, they're forecasts.

**Good Example:**
```
KR: Achieve $65K MRR (30% growth, requires new initiatives)
```

**Fix:** The "comfortable target" is your floor, not your goal. Add 30-50% stretch to what you'd hit anyway.

---

## 4. The Moonshot Mirage

**What it is:** Aspirational targets with no realistic path to achievement.

**Detection:**
- Target requires 10x or more improvement
- No initiatives identified that could drive this
- Team doesn't actually believe it's possible
- Set to impress leadership, not drive action

**Bad Example:**
```
Current users: 1,000
KR: Reach 100,000 users (100x growth in one quarter)
```

**Why it's bad:** Teams disengage from impossible goals. It creates cynicism about the goal-setting process.

**Good Example:**
```
KR: Reach 5,000 users (5x growth with identified viral loop)
```

**Fix:** Work backwards. What initiatives could drive this? If you can't name them, the target is fantasy.

## 5. Metric Dumping

**What it is:** Too many Key Results, diluting focus and accountability.

**Detection:**
- More than 4 KRs per Objective
- KRs cover multiple unrelated areas
- Team can't recall the KRs without checking documentation

**Bad Example:**
```
Objective: Grow the business
KR1: Increase signups by 20%
KR2: Reduce churn by 15%
KR3: Improve NPS by 10 points
KR4: Launch 3 new features
KR5: Increase social engagement by 50%
KR6: Reduce support tickets by 25%
KR7: Improve page load time by 40%
```

**Why it's bad:** When everything is a priority, nothing is. Attention gets spread too thin.

**Good Example:**
```
Objective: Establish product-market fit
KR1: Increase activation rate from 30% to 50%
KR2: Achieve 40% D30 retention (currently 25%)
KR3: Reach NPS of 40+ (currently 25)
```

**Fix:** Force-rank. If you could only move one metric, which would it be? Start there.

## 6. The Input Trap

**What it is:** Measuring activities (inputs) instead of results (outputs).

**Detection:**
- KRs count actions: calls made, emails sent, meetings held
- No connection between activity count and outcome
- Success is being busy, not being effective

**Bad Example:**
```
KR: Make 500 sales calls
KR: Publish 20 blog posts
KR: Send 50,000 emails
```

**Why it's bad:** You can make 500 calls and close zero deals. Activity isn't achievement.

**Good Example:**
```
KR: Close 25 new enterprise deals (from sales calls)
KR: Generate 1,000 qualified leads from content
KR: Achieve 2% email conversion rate to trial
```

**Fix:** For every input metric, identify the output it should produce. Measure the output.

---

## 7. The Rolling Goalpost

**What it is:** Regularly changing targets mid-period when they become inconvenient.

**Detection:**
- History of adjusting KRs mid-quarter
- Targets lowered when team falls behind
- "We learned and pivoted" used as excuse for misses

**Bad Example:**
```
Q1 Start: KR: Reach 10,000 users
Q1 Mid: "Market changed" -> KR: Reach 7,000 users
Q1 End: "Hit our revised target!" (7,200 users)
```

**Why it's bad:** Destroys accountability. Teams learn that targets don't matter because they'll be changed.

**Good Example:**
```
Q1 Start: KR: Reach 10,000 users
Q1 End: Achieved 7,200 users (72% of target)
Retrospective: What did we learn? What would we do differently?
```

**Fix:** Lock targets at period start. Miss honestly, learn, and adjust NEXT period.

---

## 8. Orphan Targets

**What it is:** Targets without clear individual ownership.

**Detection:**
- Owner listed as "team" or "department"
- Multiple people "share" ownership
- No single person accountable for status updates

**Bad Example:**
```
KR: Improve customer satisfaction
Owner: Customer Success Team
```

**Why it's bad:** Shared ownership means no ownership. When everyone's responsible, no one is.

**Good Example:**
```
KR: Improve NPS from 30 to 45
Owner: Sarah Chen (VP Customer Success)
```

**Fix:** One name per KR. That person updates status, flags blockers, and is accountable for outcomes.

---

## 9. The Island Effect

**What it is:** Targets that exist in isolation, disconnected from company goals.

**Detection:**
- Team targets don't map to company targets
- No explanation of how this contributes to broader goals
- Could achieve team targets while company fails

**Bad Example:**
```
Company Goal: Achieve profitability
Team KR: Increase brand awareness by 50%
(No connection to profitability)
```

**Why it's bad:** Teams optimize locally while the company fails globally. Misaligned effort.

**Good Example:**
```
Company Goal: Achieve profitability
Team KR: Reduce CAC from $100 to $70 while maintaining volume
(Directly supports profitability through efficiency)
```

**Fix:** Every team KR should answer "How does this help the company achieve [top priority]?"

---

## 10. Lagging Indicator Blindness

**What it is:** Targets that can only be measured at the end of the period.

**Detection:**
- No weekly or monthly check-ins possible
- "We'll know at quarter end" is the status update
- No early warning system for off-track goals

**Bad Example:**
```
KR: Achieve $1M ARR by Q4 end
Check-in: "We'll count revenue on December 31"
```

**Why it's bad:** By the time you know you're failing, it's too late to course-correct.

**Good Example:**
```
KR: Achieve $1M ARR by Q4 end
Leading indicators (weekly):
- Pipeline value
- Demo-to-close rate
- Average deal size
- Sales cycle length
```

**Fix:** For every lagging KR, identify 2-3 leading indicators you can track weekly.

---

## Quick Detection Checklist

Run through this for each Key Result:

| Question | Anti-Pattern if "Yes" |
|----------|----------------------|
| Could this metric improve without helping the business? | Vanity Metrics Syndrome |
| Is success defined as "shipped" not "impact"? | The Feature Factory |
| Will this definitely happen without new effort? | Sandbagging |
| Does the team believe this is impossible? | The Moonshot Mirage |
| Are there more than 4 KRs for this Objective? | Metric Dumping |
| Does this measure activity, not results? | The Input Trap |
| Has this target been lowered before? | The Rolling Goalpost |
| Is the owner a team, not a person? | Orphan Targets |
| Does this connect to company priorities? | The Island Effect |
| Can we only measure this at period end? | Lagging Indicator Blindness |
